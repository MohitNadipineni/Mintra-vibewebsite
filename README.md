# Mintra

A high-performance fashion e-commerce platform inspired by Myntra, built with Next.js App Router, Tailwind CSS, and Lucide React icons.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   # or yarn
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

The home page implements the Navbar, Hero banner, and Category grid. Follow the folder structure under `components` for layout and UI components.
